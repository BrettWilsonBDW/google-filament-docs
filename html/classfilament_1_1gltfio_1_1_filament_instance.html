Detailed Description

Provides access to a hierarchy of entities that have been instanced from a glTF asset.

Every entity has a TransformManager component, and some entities also have Name or Renderable components.

See also
AssetLoader::createInstancedAsset()

◆ getAnimator()

Animator* getAnimator ( )
noexcept

Returns the animation engine for the instance.

Note that an animator can be obtained either from an individual instance, or from the originating FilamentAsset. In the latter case, the animation frame is shared amongst all instances. If individual control is desired, users must obtain the animator from the individual instances.

The animator is owned by the asset and should not be manually deleted.
