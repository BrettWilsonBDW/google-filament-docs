◆ TransparencyMode

enum TransparencyMode : uint8_t
strong

How transparent objects are handled.

Enumerator
DEFAULT 

the transparent object is drawn honoring the raster state

TWO_PASSES_ONE_SIDE 

the transparent object is first drawn in the depth buffer, then in the color buffer, honoring the culling mode, but ignoring the depth test function

TWO_PASSES_TWO_SIDES 

the transparent object is drawn twice in the color buffer, first with back faces only, then with front faces; the culling mode is ignored.

Can be combined with two-sided lighting
