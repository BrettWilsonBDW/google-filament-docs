Specialize a constant parameter specified in the material definition with a concrete value for this material.

Once build() is called, this constant cannot be changed. Will throw an exception if the name does not match a constant specified in the material definition or if the type provided does not match.

Template Parameters
TThe type of constant parameter, either int32_t, float, or bool.
Parameters
nameThe name of the constant parameter specified in the material definition, such as "myConstant".
nameLengthLength in char of the name parameter.
valueThe value to use for the constant parameter, must match the type specified in the material definition.
