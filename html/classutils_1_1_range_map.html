Detailed Description

template<typename KeyType, typename ValueType>
class utils::RangeMap< KeyType, ValueType >

Sparse container for a series of ordered non-overlapping intervals.

RangeMap has a low memory footprint if it contains fairly homogeneous data. Internally, the intervals are automatically split and merged as elements are added or removed.

Each interval maps to an instance of ValueType, which should support cheap equality checks and copy assignment. (simple concrete types are ideal)

KeyType should support operator< because intervals are internally sorted using std::map.
