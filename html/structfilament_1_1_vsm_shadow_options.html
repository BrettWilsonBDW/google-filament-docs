Field Documentation

◆ anisotropy

uint8_t anisotropy = 0

Sets the number of anisotropic samples to use when sampling a VSM shadow map.

If greater than 0, mipmaps will automatically be generated each frame for all lights.

The number of anisotropic samples = 2 ^ vsmAnisotropy.

◆ highPrecision

bool highPrecision = false

Whether to use a 32-bits or 16-bits texture format for VSM shadow maps.

32-bits precision is rarely needed, but it does reduces light leaks as well as "fading" of the shadows in some situations. Setting highPrecision to true for a single shadow map will double the memory usage of all shadow maps.

◆ msaaSamples

uint8_t msaaSamples = 1

The number of MSAA samples to use when rendering VSM shadow maps.

Must be a power-of-two and greater than or equal to 1. A value of 1 effectively turns off MSAA. Higher values may not be available depending on the underlying hardware.
