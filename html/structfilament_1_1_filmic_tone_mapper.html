Detailed Description

"Filmic" tone mapping operator.

This tone mapper was designed to approximate the aesthetics of the ACES RRT + ODT for Rec.709 and historically Filament's default tone mapping operator. It exists only for backward compatibility purposes and is not otherwise recommended.

Maps an open domain (or "scene referred" values) color value to display domain (or "display referred") color value.

Both the input and output color values are defined in the Rec.2020 color space, with no transfer function applied ("linear Rec.2020").
